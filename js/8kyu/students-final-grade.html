Создайте функцию finalGrade, которая рассчитывает итоговую оценку учащегося в зависимости от двух параметров: оценки для экзамена и ряда завершенных проектов.

Эта функция должна принимать два аргумента: экзамен - оценка для экзамена (от 0 до 100), проекты - количество завершенных проектов (от 0 и выше);

Эта функция должна возвращать число (окончательный класс). Существует четыре типа итоговых оценок:

  • 100, если оценка для экзамена превышает 90 или если ряд завершенных проектов более 10.
  • 90, если оценка для экзамена превышает 75 и если количество завершенных проектов составляет минимум 5.
  • 75, если оценка для экзамена превышает 50 и если количество завершенных проектов составляет минимум 2.
  • 0, в других случаях

Примеры (Входы-->Вывод):

100, 12 --> 100
99, 0 --> 100
10, 15 --> 100
85, 5 --> 90
55, 3 --> 75
55, 0 --> 0
20, 2 --> 0
